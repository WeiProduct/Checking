# AI Attendance System Product Website

This folder contains the product website for the AI Attendance System.

## GitHub Pages Setup

To enable GitHub Pages:

1. Go to repository Settings
2. Navigate to Pages section
3. Under "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /docs
4. Click Save

The website will be available at: https://weiproduct.github.io/Checking/

## Local Development

To view the website locally, open `index.html` in a web browser.
